# Landing atmosphere film

Drop licensed clips here; the landing picks them up by filename via
`FilmLayer` (`src/features/marketing/FilmLayer.tsx`). Until a file exists,
the layer hides itself and the code-rendered night field carries the scene.

## Expected files

| File | Scene | Content |
|------|-------|---------|
| `hero.mp4` | Hero backdrop | Hands on strings / lamplit room, slow movement |

## Spec (per the Phase 2 decision: curated stock, graded hard)

- 3–5 candidate clips, pick the calmest take.
- 8–20s loopable, 1080p is plenty, H.264 MP4, **under ~8 MB** (it autoplays).
- Warm low-key light, near-black shadows — close textures: strings, frets,
  hands, wood grain. No faces, no logos.
- The CSS grade (`.film-grade` + video filter) unifies clips toward
  warm-lamplight-on-near-black; shoot/pick darker rather than brighter.
- License must allow commercial web use (record source + license per clip
  below when added).
